# RubySound
- 음악학원 관리 웹서버


## 활용 스킬
 - C# , ASP.NET Core
 

## 주요 기능
- 웹사이트 계정 등록 및 로그인
- 선생님 / 수강생 조회 및 관리
- 게시판 글 작성 및 삭제
